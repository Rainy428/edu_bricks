# edu_bricks

教育用途向けの Luanti / Minetest 用 保護ノードMODです。

生徒が誤って壊せないブロックを提供しつつ、
教師は編集・回収が可能な設計になっています。

授業用ワールドを想定して作られています。

---

## 機能

* ステージ用保護ブロック
* 編集可能な保護看板
* 保護された松明（床・壁・天井対応）
* 教師権限（privilege）システム
* 薄いノード回収時のクリック貫通防止

生徒はこれらのノードを破壊できません。
教師のみ操作できます。

---

## ノード一覧

### ステージブロック

授業やステージ構築用の保護ブロックです。

生徒は破壊できません。
教師は回収できます。

---

### 看板

編集可能な保護看板です。

操作方法：

左クリック
→ 看板の内容を表示

AUX1 + 左クリック
→ 看板を編集（教師のみ）

SHIFT + AUX1 + 左クリック
→ 看板を回収（教師のみ）

最大文字数：512文字

---

### 松明

保護された松明です。

以下の設置に対応：

* 床
* 壁
* 天井

回収方法：

SHIFT + AUX1 + 左クリック

---

## 重要（モデルについて）

このMODは、デフォルトゲームに含まれる以下のモデルを使用します：

* `torch_floor.obj`
* `torch_wall.obj`
* `torch_ceiling.obj`

これらのファイルは **このMODには含まれていません**。

Minetest Game 以外のゲームを使用している場合、
松明が正しく表示されない可能性があります。

---

## 権限（Privilege）

教師には以下の権限が必要です：

```id="h9kq24"
edu_teacher
```

付与例：

```id="l1n4v7"
/grant プレイヤー名 edu_teacher
```

シングルプレイヤーでは自動的に付与されます。

---

## インストール方法

MODフォルダを以下に配置：

```id="r3c1m8"
mods/
```

例：

```id="z0b9gq"
worlds/
 └─ yourworld/
     └─ worldmods/
         └─ edu_bricks/
```

ゲームを再起動してください。
